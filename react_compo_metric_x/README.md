# Titre du Projet

Description courte du projet.

## Prérequis

Avant de commencer, assurez-vous d'avoir installé :
- Node.js
- npm ou yarn

## Installation

Suivez ces étapes pour installer le projet :

1. Clonez le dépôt : 
```code 
git clone https://grond.iut-fbleau.fr/bade/ReactCompoMetricsX.git
```
2. Accédez au répertoire du projet :

```code
cd le_nom_du_depot
```

3. Installez les dépendances :

```code
cd front 
npm i
```

```code
cd back 

cp .env.example .env

npm i
```


## Démarrage du projet

Pour démarrer le serveur de développement, exécutez :

```code
cd back 

npm start 

Puis 

cd front 

npm start 
```


Le serveur de développement sera accessible à l'adresse [http://localhost:3000](http://localhost:3000).
